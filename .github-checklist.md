## Pre-commit Checklist

Before pushing to GitHub:

1. **Verify .gitignore**
   ```bash
   git status --ignored
   ```
   Ensure `bag.sqlite` and `lvbag-extract-nl.zip` are ignored

2. **Check repository size**
   ```bash
   du -sh .git
   ```
   Should be ~25 MB or less

3. **Test sample database**
   ```bash
   python3 test-sample-db.py
   ```

4. **Verify documentation**
   - README.md is complete
   - Links work
   - Examples are accurate

5. **Environment files**
   - `.env` is NOT committed (only .env.example, .env.sample, .env.production)
   - `.env.example` has all variables documented

## Quick Start (for GitHub users)

After cloning the repository:

```bash
# Install dependencies
pip install -r requirements.txt

# Configure for sample database
cp .env.sample .env

# Run API
uvicorn main:app --host 0.0.0.0 --port 7777 --reload

# Or with Docker
docker-compose up -d
```
